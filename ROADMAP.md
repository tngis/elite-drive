# Машин түрээсийн платформ — Phase-д хуваасан Roadmap

> Хоёр талт зах зээл (two-sided marketplace): **түрээслэгч (renter)** ↔ **түрээслүүлэгч (owner)**
> Зорилго: Монголын хамгийн том машин түрээсийн платформ, өдөр тутмын хэрэглээ.
> Нийт **4 phase**. Доод phase нь дээд phase-ийн суурь болно.

---

## Phase 1 — MVP (Суурь зах зээл ажиллаж эхлэх)

> Зорилго: Хүн машин нийтэлж, өөр хүн хайж олоод түрээслэх хүсэлт явуулж, гар аргаар тохиролцоод гүйлгээ хийж чадах хамгийн бага бүтэн систем.

### Account & Identity
- Бүртгүүлэх (утас/имэйл)
- Нэвтрэх / гарах / нууц үг сэргээх
- Утасны дугаар OTP баталгаажуулах
- Нэг аккаунт дотор **renter ба owner** хоёр үүрэг
- Профайл засах (зураг, нэр, холбоо барих)

### Машин бүртгүүлэх (Owner)
- Шинэ машин нэмэх (дугаар, марк, модель, он, өнгө)
- Олон зураг оруулах
- Тоноглол / онцлог сонгох
- Үнэ тогтоох (өдрийн үнэ + deposit)
- Машин засах / идэвхгүй болгох / устгах
- Боломжтой/боломжгүй огноо (энгийн calendar)

### Машин хайх (Renter)
- Машин жагсаалт (list/grid)
- Үндсэн шүүлтүүр: байршил, огноо, үнэ, ангилал, хроб/автомат
- Эрэмбэлэх: үнэ, шинэ
- Машины дэлгэрэнгүй хуудас (зураг, тоноглол, үнэ)
- Огноогоор боломжтойг шалгах

### Захиалга (Booking)
- Түрээслэх **хүсэлт илгээх** (огноо, байршил)
- Үнийн задаргаа урьдчилан харах
- Owner **зөвшөөрөх / татгалзах**
- Захиалгын статус: pending → approved → active → completed / cancelled
- Захиалга цуцлах
- Захиалгын түүх (хоёр талд)

### Төлбөр (одоогийн, гар арга)
- Банкны шилжүүлэг / QPay / SocialPay (QR)
- Бэлэн сонголт + платформ дээр баталгаажуулах
- Deposit гар аргаар хяналт

### Мэдэгдэл (хамгийн бага)
- In-app мэдэгдэл + SMS чухал үйлдэлд (OTP, хүсэлт, зөвшөөрөл)

### Дэд бүтэц
- Мобайл-эхэлсэн responsive дизайн
- Зураг cloud storage
- Энгийн админ: хэрэглэгч/машин/захиалга харах, блоклох

## Phase 2 — Найдвартай гүйлгээ (Trust + бүрэн захиалгын мөчлөг)

> Зорилго: Хүлээлгэн өгөх, буцаах, хугацаа, алданги, итгэлийн механизм нэмж бодит ажиллагааг бүрэн болгох.

### Хүлээлгэн өгөх (Handover)
- Авах үеийн чек-лист (4 тал зураг, км, түлш)
- Буцаах үеийн чек-лист (харьцуулалт)
- Хоёр талын тоон баталгаажуулалт (digital handover)
- Гэмтэл/тэмдэглэл бүртгэх

### Хугацаа сунгах (Extension)
- Идэвхтэй захиалга дээр **сунгах хүсэлт**
- Owner зөвшөөрөх + давхцал шалгах
- Нэмэлт төлбөр тооцох
- **Сунгах deadline** (буцаахаас X цагийн өмнө хүсэлт гаргах)

### Хоцролт ба алданги (Late & penalty)
- Буцаах хугацаа хэтэрсэнийг автоматаар илрүүлэх
- **Нэмэлт алданги** (цаг/хоногоор тооцох дүрэм)
- Алдангийг эцсийн төлбөр / deposit-оос суутгах
- Гэмтэл/цэвэрлэгээ/түлшний нөхөн төлбөр

### Мэдэгдэл (өргөтгөсөн)
- Push notification (мобайл)
- Trigger: авах/буцаах сануулга, **буцаах ойртсон**, **сунгах deadline ойртсон**, хоцролт, шинэ review
- Мэдэгдлийн тохиргоо

### Итгэл ба үнэлгээ
- **Хоёр талын review** (од + текст), зөвхөн дууссан захиалгад
- Дундаж зэрэглэл харуулах
- KYC: жолооны үнэмлэх / иргэний үнэмлэх баталгаажуулалт
- Тайлагнах (report)

### Харилцаа
- Renter ↔ owner чат (захиалгатай холбоотой)

## Phase 3 — Ухаалаг туршлага + бодит онлайн төлбөр (Scale & UX)

> Зорилго: Хэрэглэгчийн туршлагыг сайжруулж, төлбөрийг автоматжуулж, платформын орлогын механизмыг бэхжүүлэх.

### Машин санал болгох (Recommendation)
- Хувийн санал (хайлт/захиалгад тулгуурласан)
- "Ойролцоо байгаа машинууд" + газрын зураг (map view)
- "Их захиалгатай / шилдэг үнэлгээтэй" / featured
- Ижил төстэй машин (детайл хуудсанд)
- Favorites / wishlist, сүүлд үзсэн

### Хайлт сайжруулах
- Газрын зураг дээр хайх (geolocation)
- Илүү нарийн шүүлтүүр (түлш, суудал, он, тоноглол)
- Instant booking горим (зөвшөөрөл хүлээхгүй)

### Төлбөр (ideal, автомат)
- Карт холбох, онлайн төлбөр
- **Escrow** — мөнгийг платформ түр барих
- Автомат deposit hold & release
- Платформын шимтгэл автомат суутгал (commission split)
- Owner-д автомат payout
- Invoice / receipt / refund
- Гүйлгээний түүх, санхүүгийн самбар

### Маргаан ба аюулгүй байдал
- Маргаан шийдвэрлэх (dispute) процесс
- Даатгалын мэдээлэл харуулах
- Blacklist

## Phase 4 — Платформ хэмжээ ба эрх мэдэл (Growth, fleet, analytics)

> Зорилго: Олон машинтай эзэд, бизнес хэрэглэгч, гүн анализ, өргөтгөл.

### Fleet / бизнес owner
- Олон машин нэг эзэн удирдах (multi-listing dashboard)
- Багаар удирдах эрх (бизнес аккаунт)
- Динамик үнэ тогтоолт (улирал/эрэлтээр)

### Админ ба анализ
- Бүрэн analytics dashboard (орлого, ашиглалт, conversion)
- Алданги/үнийн дүрэм админаар тохируулах
- Контент удирдлага (featured, ангилал)
- Payout / шимтгэлийн дэлгэрэнгүй удирдлага

### Өргөтгөсөн боломжууд
- Олон хэл (Монгол/Англи)
- Mobile app (native) бүрэн хувилбар
- Лояалти / урамшуулал / promo код
- Referral систем
- Маск дугаар / платформоор дамжсан дуудлага
- API интеграци (даатгал, банк, төрийн бүртгэл)

---

## Phase-уудын товч логик

| Phase | Гол санаа | Үр дүн |
|-------|-----------|--------|
| **1** | Суурь зах зээл | Хүн машин нийтэлж, хайж, гар аргаар түрээслэнэ |
| **2** | Найдвартай гүйлгээ | Handover, сунгалт, алданги, review, итгэл |
| **3** | Ухаалаг + онлайн төлбөр | Recommendation, escrow, автомат тооцоо |
| **4** | Хэмжээ ба өсөлт | Fleet, analytics, app, өргөтгөл |
